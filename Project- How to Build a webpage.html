Build out your HTML. HTML is your bones for your page. It is the structure that will hold all the skin and brains together (skin=css, brains=JS)

Step Two

The next step is using CSS. CSS is the skin and decoration for your page.

Some forms of CSS code would look like this:

color: red;
}
